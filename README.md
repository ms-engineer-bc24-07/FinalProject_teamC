# Connetto
社内のコミュニケーションの場を自動で設定してくれるアプリ<br>
法人が福利厚生ツールとして契約。<br>
利用者は、飲み会など参加希望の日時や希望条件を登録するだけで、飲み会の場所や日時を自動で提案・決定します。<br>
これにより、幹事の負担を大幅に軽減し、飲み会を迅速にセットアップできます。
